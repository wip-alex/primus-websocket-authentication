# primus-websocket-authentication
---------------------------------

This is an evaluation of websocket authorization to accept or refuse
incoming connections using JSON Web Token (JWT) sent as a query string parameter.

### USE
- start server
```
npm install
npm start
```
- start client
```
npm run client
```
